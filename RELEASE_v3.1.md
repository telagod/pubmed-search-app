# v3.1 最佳实践版发布说明

## 亮点
- 数据完全本地化：上传/下载数据库，云端零存储占用
- 新增首页「💾 数据管理」：上传、查看、下载、清空临时数据
- 深色模式默认启用：长时间阅读更舒适
- 页面与流程优化：更贴合真实使用步骤
- 稳定性与体验提升：缓存与临时库策略优化

## 变更详情
- UI/UX：深色主题、首页数据管理入口、侧边栏信息增强
- 数据：临时 SQLite 库 + 显式导入导出，避免云端持久化依赖
- 缓存：使用 `st.cache_resource` 优化资源复用
- 结构：页面顺序调整（数据管理 → Dashboard → 高级搜索 → 文献浏览 → 数据分析 → 设置 → 关于）

## 兼容性与升级指引（从 v3.0 升级）
1. 旧数据迁移：将 v3.0 导出的数据库文件在首页「💾 数据管理」中上传
2. 继续使用：上传后可立即浏览与分析，可继续检索并再次下载更新后的数据库
3. 新部署：先在「⚙️ 设置」配置 PubMed 邮箱与 API Key，再在「🔍 高级搜索」执行检索，完成后下载数据库

## 安装与运行
```bash
# 使用 uv 建议
uv venv
source .venv/bin/activate  # Windows: .venv\Scripts\activate
uv pip install -r requirements.txt

# 启动应用
uv run streamlit run streamlit_app.py
```

## 校验清单
- 首页显示「💾 数据管理」，可上传/下载数据库
- 侧边栏与页脚显示 v3.1 版本信息
- 页面默认深色主题
- 执行一次检索并下载数据库；刷新后上传数据库应可无缝继续
